Compile:

```
$ rustc <file_name>.rs
```

Execute:

```
$ ./<file_name>
```

using Cargo:

initialize project:

```
$ cargo new <project_name>
```

build project:

```
$ cargo build
```

build project for production:

```
$ cargo build --release
```

build and run project:

```
$ cargo run
```
